INSTRUCCIONES PARA ACCESO REMOTO CON NGROK
==============================================

1. INICIAR EL TUNEL
-------------------
Abrir PowerShell y ejecutar:

& "C:\Users\user1\AppData\Local\Microsoft\WinGet\Packages\Ngrok.Ngrok_Microsoft.Winget.Source_8wekyb3d8bbwe\ngrok.exe" http 80


2. OBTENER LA URL PUBLICA
--------------------------
Despues de ejecutar el comando, buscar la linea "Forwarding" en la terminal.
La URL termina en .ngrok-free.dev

Ejemplo:
Forwarding    https://gillian-nonappropriable-nonemulously.ngrok-free.dev -> http://localhost:80

Esa URL es la que se usa en cualquier dispositivo para acceder al sistema.


3. ACCEDER DESDE CELULAR U OTRO DISPOSITIVO
--------------------------------------------
- Abrir navegador
- Ingresar la URL completa (con https://)
- Primera vez: hacer clic en "Visit Site" cuando aparezca advertencia de ngrok
- El sistema redirige automaticamente a /Sistem Hotel Cecil/


4. DETENER EL TUNEL
--------------------
Presionar Ctrl + C en la terminal donde esta corriendo ngrok.


5. NOTAS IMPORTANTES
---------------------
- La terminal debe permanecer abierta mientras se necesite acceso remoto
- Si se cierra la terminal, el tunel se desactiva
- La URL cambia cada vez que se reinicia ngrok (cuenta gratuita)
- XAMPP/Apache debe estar corriendo en el puerto 80
- El archivo C:\xampp\htdocs\index.php redirige automaticamente al proyecto


6. VERIFICAR QUE TODO FUNCIONA
-------------------------------
- Apache debe estar activo en XAMPP
- MySQL debe estar activo en XAMPP
- El tunel ngrok debe estar corriendo
- Probar primero desde la PC con la URL de ngrok
- Luego probar desde el celular


7. SOLUCIONAR PROBLEMAS
------------------------
Si aparece error ERR_NGROK_3200:
- Verificar que Apache este corriendo
- Verificar que este en el puerto 80
- Reiniciar ngrok

Si no carga en celular:
- Verificar que la URL este completa (con https://)
- Hacer clic en "Visit Site" si aparece advertencia
- Verificar conexion a internet del celular
